# 常见的问题

## 服务器相关

### 为什么岛屿群系无法更改？

答：在岛屿群系界面不要直接使用右键，可以先右键群系选择按钮，以区块的方式更改群系

答：此问题描述不够详细，因此我给出以下几个原因，请一一排查
