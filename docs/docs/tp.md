# 关于传送

岛屿传送点(岛屿地标)在空岛板块介绍过了，感兴趣可以去翻阅

**内容较少，指令简介不做大篇幅介绍**

## 传送相关指令

`/tpa <玩家>` #传送到玩家<br/>
`/tpahere <玩家>` #将玩家传送到你这<br/>
`/sethome <名字>` #设置家，默认可设置3各<br/>
`/home <名字>` #传送到家<br/>
`/homes` #打开家列表<br/>
`/delhome <名字>` #删除家<br/>
`/homelist` #家列表<br/>
`/warps` #打开地标菜单<br/>
`/spawn` #回到主城<br/>
`/back` #返回上一次死亡或使用传送的位置<br/>
